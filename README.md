# QuanLyQuanAn (Visual studio 2019 or 2022)
Hướng dẫn chạy được phần mềm:
- Tải về và giải nén file .rar
- Chạy file .sql
- Thay đổi đường dẫn csdl tại Methods/XuLy_CauTruyVan.cs
- Run phần mềm
